# sesarengan

A modern, mobile-first wedding invitation dashboard built with Laravel, Alpine.js, and Tailwind CSS.

## Features
- Mobile-first tabbed dashboard interface.
- Real-time guest list management with WhatsApp integration.
- Responsive live preview system.
- Premium luxury invitation templates.
- Persistent data storage using PostgreSQL & Supabase.

## Deployment
Deployed on Vercel with serverless PHP.
